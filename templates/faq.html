Frequently Asked Questions

What is Playarr?

Playarr is a tool to scan and fix media files for compatibility with media clients like Plex, Jellyfin, and Emby.

How do I start a scan?

Select a client profile and click the "Start Scan" button on the home page.

Can Playarr fix my media files?

Yes, Playarr can transcode audio tracks for files that need attention to improve compatibility.
